# BrainNote Self Report

## 目的

BrainNote自身が完成したとき、BrainNoteをReportBoxへ提出し、Knowledge / Component / Function Ticketへ昇格できるようにする。

## Report対象

- アプリ名: BrainNote
- 役割: Report / IDIA / Knowledge / Memory / Goals / Workspace Boardを集約するWorkspace PWA
- 実装: SolidJS + Vite + PWA

## 画面

- Dashboard
- ReportBox
- IDIA Inbox
- Workspace Board
- Goals
- Dev TODO
- Knowledge Index
- Memory_0610
- Function Ticket

## DONE定義

- PWAで閲覧・軽い編集ができる
- ReportからKnowledgeへ昇格できる
- IDIAを本番開発TODOと分けて管理できる
- Workspace BoardでCodex相談・資料・PWAメモを割り振れる
- JSON / MD Exportができる

## Knowledge候補

- Workspace Dashboardと開発TODOは分ける
- 荒いアイディアはIDIA Inboxにため、タグとゲートで整理する
- Codex相談用の資料はmdテキストとlocal pathをセットで残す
- Tauri DesktopではPWAメモをlocalフォルダにまとめる
